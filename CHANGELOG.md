## 0.1.0

- Initial release
- `Shimmerize` wrapper widget with auto-detection of `Text`, `RichText`, `Icon`, `Image`, and `CircleAvatar`
- `Bone` widget for explicit skeleton placeholders with `Bone.circle` constructor
- `ShimmerTheme` and `ShimmerThemeData` for customizable shimmer appearance
- Animated gradient shimmer effect via `ShimmerPainter`
- Zero external dependencies
